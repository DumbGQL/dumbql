@dumbql/core

Stable Angular 22+

The core package is the only required dependency. It provides the GraphqlService, middleware pipeline, the gql tagged template literal, standalone utility functions, pipes, reactive variables, a client directive, router integration, auto-refetch, and configuration.

Size: ~3.2 kB min+gzip Dependencies: graphql, @angular/core

Installation

npm install @dumbql/core
@dumbql/core is automatically included when you install any @dumbql/* package — it's listed as a peer dependency.
