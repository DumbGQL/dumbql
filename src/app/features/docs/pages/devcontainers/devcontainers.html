Dev Containers

Spin up a fully-configured development environment with DumbQL in seconds.

Each devcontainer includes a running mock GraphQL backend and a minimal app that queries it using @dumbql/* packages — zero setup required.

Prerequisites

  • VS Code with the Dev Containers extension
  • Docker

Angular Demo

The full DumbQL monorepo — Angular 22 app with cache, subscriptions, file uploads, pagination, and all other packages.

How It Works

Each devcontainer uses the mcr.microsoft.com/devcontainers/typescript-node:22 base image. The postCreateCommand installs dependencies and starts both the mock backend and the dev server. The mock server (in mock/) provides a GraphQL endpoint with sample notes and users.

Source Code

Demo apps live in the demos/ directory:

  • demos/react/ — React + Vite demo
  • demos/vue/ — Vue + Vite demo

For Angular, the monorepo itself (src/) serves as the demo app.
