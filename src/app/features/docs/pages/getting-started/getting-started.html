Starters

Kickstart a new project with one of our starters:

  • Angular 22+ — Open in StackBlitz · Open in Codespaces · git clone + cd starters/angular && npm start — standalone app with @dumbql/client, mock backend, and provideDumbql + DumbqlQuery directive.
  • React 18+ — Open in StackBlitz · git clone + cd starters/react && npm start — standalone app with @dumbql/client, @dumbql/react, mock backend, and useQuery hook.
  • Vue 3+ — Open in StackBlitz · git clone + cd starters/vue && npm start — standalone app with @dumbql/client, @dumbql/vue, mock backend, and useQuery composable.
