@dumbql/middlewares

Stable Angular 22+

The middlewares package provides pre-built middleware functions that plug into the DumbQL middleware pipeline. Each middleware is a pure function that wraps the next link in the chain, giving you full control over request/response flow.

authRefresh

Automatically attaches an authorization header and refreshes the token when a 401 response is received: 

{{ authRefreshCode }}

retryExchange

Retries failed operations with configurable delay and attempt limits: 

{{ retryExchangeCode }}

focusRefetch

Refetches active queries when the browser tab regains focus: 

{{ focusRefetchCode }}

offlineQueue

Queues mutations when the browser is offline and replays them when connectivity returns: 

{{ offlineQueueCode }}

Composing Middlewares

Combine multiple middlewares using composeMiddlewares from @dumbql/core. The order determines execution order:
