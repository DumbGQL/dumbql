Why DumbQL?

Existing GraphQL clients for Angular are either abandoned ports of React libraries (Apollo Angular), overly complex (Relay), or too simplistic. DumbQL is designed from the ground up for Angular's DI system, leverages signals for reactivity, and keeps each package small enough that the entire library adds less than 25 kB to your production bundle.

  • Zero magic — explicit middleware pipeline, no hidden cache layers
  • Full control — configure every aspect from cache policies to retry logic
  • Modular — 12 independent packages, import only what you use
  • Signals-native — built for Angular's reactive primitives
  • SSR ready — hydration, transfer state, chunked transfer
  • Strongly typed — full TypeScript inference from queries
