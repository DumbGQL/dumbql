@dumbql/subscriptions

Stable Angular 22+

The subscriptions package enables real-time data streaming over WebSocket using the graphql-transport-ws protocol. It integrates fully with the DumbQL middleware pipeline so authenticated subscriptions work out of the box.

graphql-transport-ws Protocol

DumbQL subscriptions implement the graphql-transport-ws protocol (the modern replacement for subscriptions-transport-ws). This provides:

  • Single connection multiplexing — one WebSocket handles many subscriptions
  • Auto-reconnect — with exponential backoff and jitter
  • Connection params — send auth tokens during the connection_init handshake
  • Complete / Error handling — per-subscription lifecycle callbacks
